## Welcome to Your Blog

This is a sample blog post. You can delete this file and create your own posts.

## How to Create Posts

Create new markdown files in:
- `src/content/blog/en/` for English posts
- `src/content/blog/ja/` for Japanese posts

### Frontmatter

```yaml
---
title: 'Your Post Title'
description: 'A brief description for SEO'
pubDate: 2024-01-01
tags: ['tag1', 'tag2']
draft: false
---
```

## Markdown Features

You can use all standard markdown features including code blocks, lists, and more.

```javascript
console.log('Hello, World!');
```
